# Minecraft Item Texture Sources

Item, block, and shield entity texture PNGs in this folder were downloaded from the Minecraft 1.20.1 assets mirror:

https://github.com/InventivetalentDev/minecraft-assets/tree/1.20.1/assets/minecraft/textures

The site uses these textures only as item sprites for the event shop UI and does not include Minecraft, Mojang, Microsoft, or other official logos.
